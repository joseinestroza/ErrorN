Bootstrap CSS & JS

You must include Bootstrap css and js, whether local or online. If the link below becomes outdated, you can go to getbootstrap.com to get the current link. At the time of publishing, the supported version of Bootstrap is 5.3.2 .

You must include the ErrorN JavaScript code, whether local or online; this one is the second script component shown below. Download it from here: ErrorN.min.js.

    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-T3c6CoIi6uLrA9TneNEoa7RxnatzjcDSCmG1MXxSR1GAsXEV/Dwwykc2MPK8M2HN" crossorigin="anonymous">
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js" integrity="sha384-C6RzsynM9kWDrMNeT87bh95OGNyZPhcTNXj1NW7RuBCsyN/o0jlpcV8Qyq46cDfL" crossorigin="anonymous"></script>
    <script src="ErrorN.min.js"></script>

Code Snippets

Copy the snippet you need for your specific case.

Locate the notification toast in the upper right corner

/*Locate the notification toast in the upper right corner. This is the default value when no "setPosition" method is called.*/
ErrorN.setPositionTopRight();

Locate the notification toast in the upper left corner

/*Locate the notification toast in the upper left corner.*/
ErrorN.setPositionTopLeft();

Locate the notification toast in the lower right corner

/*Locate the notification toast in the lower right corner.*/
ErrorN.setPositionBottomRight();

Locate the notification toast in the lower left corner

/*Locate the notification toast in the lower left corner.*/
ErrorN.setPositionBottomLeft();

Locate the notification toast in the center of the screen

/*Locate the notification toast in the center.*/
ErrorN.setPositionCenter();

Display an error notification

/*Displays an error notification using a simple 
format without the duration of the message on the screen.*/
ErrorN.error("message");
/*Displays an error notification using a simple 
format with the duration of the message on the screen, 
in milliseconds (9000 is the default).*/
ErrorN.error("message", 9000);

                
/*Displays an error notification using a complex 
  format without the duration of the message on the screen.*/
  ErrorN.error("message", "main title", "small text");

                  
/*Displays an error notification using a complex 
  format with the duration of the message on the screen, 
  in milliseconds (9000 is the default).*/
  ErrorN.error("message", "main title", "small text", 9000);
